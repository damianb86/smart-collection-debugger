# Product brief

## Problema
Merchants struggle to understand smart collection rules, especially when variant-sensitive rules behave unexpectedly.

## Usuario objetivo
Catalog manager maintaining automated collections and launch collections.

## Merchant persona
Merchant con catalogo activo, equipo chico y necesidad de resolver un problema operativo puntual sin instalar una suite pesada.

## Flujo principal
Pick a product and collection, inspect each rule, identify the first failing condition, then fix product data.

## Propuesta de valor
A simple explainability layer for collection rules that turns opaque matching into a readable checklist.

## MVP
- Dashboard with collections needing review.
- Rule debugger table with pass/fail/explain status.
- Filter by product, collection and failing rule type.
- Empty state for stores with no automated collections.

## Futuro fuera de alcance
- Admin product and collection UI extensions.
- Bulk collection drift report.
- Suggested product data fixes.
